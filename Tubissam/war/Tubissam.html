Descripcion

La aplicación web Tubissam se encargará de proporcionar a los internautas información útil y actualizada para el uso de transportes públicos en Sevilla Capital, concretamente autobús y bicicleta

En lo relativo al transporte de autobús se usará la api de Tussam para aportar información sobre tiempos de llegada, situación de las paradas en el mapa, autobuses que pasan por ellas etc.

En cuanto al transporte en bicicleta se usará la api de citybik para aportar información sobre la situación y el estado de las estaciones de Sevici (puestos libres), comunicados por carriles bici.

También se incorporará la predicción meteorológica próxima, especialmente útil para los usuarios de Sevici. Para ello se usará la api Weather.

Además, mediante el registro a través de la cuenta de google, se puede guardar información adicional más personalizada sobre estaciones favoritas o buses habituales.

Por último añadir que la aplicación consumirá feeds de datos de Tussam utilizando el formato RSS.
